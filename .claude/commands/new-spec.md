# /new-spec

## 目的
新しい機能の仕様書（PRD.md）を作成する。

## 手順

1. ユーザーから機能名を受け取る（引数として渡される）
2. `~/.claude/templates/prd-template.md` を読み込む
3. 以下を私に質問する：
   - 「なぜこの機能が必要ですか？」
   - 「完成した状態をどう定義しますか？」
4. 回答をもとに `~/.claude/specs/[機能名]/PRD.md` を作成する
5. タスクを3〜5個に分解して PRD.md の「タスク一覧」に書く
6. 作成完了後、タスク一覧を表示して確認を求める

## 出力ファイル
- `~/.claude/specs/[機能名]/PRD.md`
